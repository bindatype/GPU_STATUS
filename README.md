# GPU_STATUS
extract GPU stats in a more ETL friendly way than nvidia-smi. btw, nvidia-smi has an xml output and if that works you 
the by all means use that instead :)

## Dependencies


| Lib  | Version |
| ------------- | ------------- |
| CUDA  | 12.2 or greater  |
| NVIDIA  | 545.23.08 or greater  |
